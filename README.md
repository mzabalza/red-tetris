## SOCKET.IO 

### Usage
```
cd src/client
npm run dev-server
```
```
cd src/server
npm run dev
```

http://localhost:8080/

### Star with Docker in MAC
```
docker-machine create --driver virtualbox default (First time)
docker-machine start default (After first time)
eval $(docker-machine env default)
```
